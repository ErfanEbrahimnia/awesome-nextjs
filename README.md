## Auth / Authz

- [Better Auth](https://github.com/better-auth/better-auth) – The most comprehensive authentication framework for TypeScript
- [Kilpi](https://github.com/jussinevavuori/kilpi) – Kilpi is the open-source TypeScript authorization library designed for developers who need flexible, powerful, and intuitive authorization.
- [iron-session](https://github.com/vvo/iron-session) – iron-session is a secure, stateless, and cookie-based session library for JavaScript.

## Component Libraries

- [shadcn/ui](https://github.com/shadcn/ui) – A set of beautifully-designed, accessible components and a code distribution platform. Works with your favorite frameworks. Open Source. Open Code.
- [Elevenlabs UI](https://github.com/elevenlabs/ui) – ElevenLabs UI is a component library and custom registry built on top of shadcn/ui to help you build multimodal agents faster.
- [Auth CN](https://github.com/MrUprizing/auth-cn) – Full-stack Components for Better Auth using shadcn/registry.

## File Upload

- [Better Upload](https://github.com/Nic13Gamer/better-upload) – Simple and easy file uploads for React. Upload directly to any S3-compatible service with minimal setup.

## State Management

- [nuqs](https://github.com/47ng/nuqs) – Type-safe search params state manager for React frameworks - Like useState, but stored in the URL query string.
- [Zustand](https://github.com/pmndrs/zustand) – A small, fast and scalable bearbones state-management solution using simplified flux principles. Has a comfy API based on hooks, isn't boilerplatey or opinionated.

## Templates

- [OpenStatus Template](https://github.com/openstatusHQ/openstatus-template) – We've created this template to help you get started with your @shadcn/ui project. It uses @nextjs in an SPA mode and can be exported statically (BYO router).

## Resources

- [Next.js Weekly](https://nextjsweekly.com/) – A weekly newsletter with the most interesting Next.js News, Tutorials, Projects, and Tools. The easiest way to keep up with what’s happening in the ecosystem.

## Other

- [Anzen](https://github.com/SugarDarius/anzen) – Fast, flexible, framework validation agnostic, type‑safe factories for creating route handlers, page and layout Server Component files in Next.js.
- [Next SEO](https://github.com/garmeeh/next-seo) – Next SEO is a plug in that makes managing your SEO easier in Next.js projects.
